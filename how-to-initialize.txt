1. Initialize a local Git repository:
 git init
2. Add files:
 git add .
3. Commit changes:
 git commit -m "Initial commit"
4. Create a new empty repository on GitHub webpage (no README).
5. Connect to GitHub via SSH:
 git remote add origin git@github.com:franciscospauldingastudillo/simple-adder.git
6. Push to GitHub:
 git push -u origin main
